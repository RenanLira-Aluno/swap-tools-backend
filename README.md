# Api Swap Tools

## Requisitos
  - Docker
  - NodeJs >= 20

## Instalação
```bash
  git clone git@github.com:RenanLira-Aluno/swap-tools-backend.git
  cd swap-tools-backend

  npm install

```

## Para Rodar o Projeto
```bash
  docker compose -f ./docker-compose.yml -d
  npm run start # or npm run start:dev
```
